Tags

tags are main elements of html. We use tags for give an structure the content.

  • html: Bir web sayfasının ana elemanı
  • head: Döküman hakkında bilgiler içerir
  • title: Dökümanın başlığı
  • h1-h6: Başlıklar
  • p: Paragraf
  • a: Link
  • ul-ol: Listeler
  • div-span: Generic containers
  • form-input: Formlar ve girdiler

META Tags

Döküman hakkında ek bilgiler verir. Web sayfasında direk olarak gözükmez. Bazı metag taglar aşağıda verilmiştir:

  • charset="UTF-8" -> Karakter kodlamasını belirtir
  • viewport -> farklı ekran boyutları için layoutları kontrol eder
  • description -> Arama sonuçlarında gözükür ve 160 karakterden fazla olmamalı
  • keywords -> Dökümanın içeriği hakkında bilgi verir
  • author -> Yazarın ismi
  • robots -> Arama motorlarının sayfayı indexlemesi hakkında bilgi verir
    • index/noindex -> indekslenip indekslenmeyeceğini belirtir
    • follows/nofollows -> sayfadaki linklerin takip edilip edilmeyeceği
